Detailed Description

OpenRPNCalc Operating System (ORCOS) library header.

This is a subset of functions from dmcp.h, distributed under BSD-3 license. Provides hardware abstraction and core functionality for OpenRPNCalc. Original source: https://github.com/swissmicros/DMCP5_SDK/blob/master/dmcp/dmcp.h
